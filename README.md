# ReNum

Plugin for Sublime Text 3 which renumbers lines with specified pattern.

For example:  

    {/*1*/ block of code}
    {/*1*/ block of code}
    ...
    {/*1*/ block of code}

will be changed to:  

    {/*0*/ block of code}
    {/*1*/ block of code}
    ...
    {/*N*/ block of code}

## Features
- keeps leading zeros
